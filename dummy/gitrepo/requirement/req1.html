1.1.1. User characteristics

A user will have the following attributes:

  • email

  • password

  • role

  • first name

  • last name

  • company name

  • location

  • phone number

  • fax number

  • signature (picture or just a string)

  • profile picture

  • account status

    • last connection timestamp

    • creation date

    • logged in

    • enabled

    • expired

    • locked

    • credentials expired
